To Install:

1) Replace build-win32.cmd and win32-init_env_command.cmd in the Openvibe\scripts folder with the ones in this folder

2) Place the folder space-invaders in openvibe\openvibe-applications\vr-demo\trunc\share\openvibe-applications\vr-demo

3) Place the sourcefiles from Brain Invaders\Brain Invaders in openvibe\openvibe-applications\vr-demo\trunc\src

4) Compile using Openvibe\scripts\win32-build.cmd

5) Run ov-vr-demo-space-invader.cmd in openvibe\dist.
